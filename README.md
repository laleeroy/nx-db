# Tinfoil Metadata Collection

This repository contains scripts for managing Tinfoil metadata.

## Installation

To install the required dependencies for the `auto_add` script, run:

```bash
pip install -r requirements.txt
```

Getting the Title Database

The primary metadata file (titles.json) can be downloaded from the official Tinfoil repository:

https://tinfoil.media/repo/db/titles.json

Usage

After installing the dependencies and obtaining the titles.json file, you can run the auto_add script to process the metadata.
